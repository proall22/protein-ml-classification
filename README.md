\# Machine Learning for Protein Function Classification



This project applies supervised machine learning to classify proteins based on numerical features derived from amino acid sequences.



\## Tools

\- Python

\- Pandas

\- Scikit-learn

\- Matplotlib



\## Method

\- Random Forest classifier

\- Train/test split

\- Standard evaluation metrics



\## Applications

\- Bioinformatics

\- Drug discovery

\- Disease-related protein analysis
